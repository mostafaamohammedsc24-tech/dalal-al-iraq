ممتاز جداً! الموقع عاد للحياة، وهذه أفضل لحظة لبناء الأساس الصلب. سأعطيك قائمتين منفصلتين ودقيقتين: واحدة للمكاتب والأخرى للمحامين. كل قائمة مصممة كمواصفات برمجية جاهزة لتطبيقها في لوحة التحكم.

أولاً: قائمة ميزات المكاتب العقارية

أ. التسجيل والدخول

الميزة الوصف التقني
تسجيل مكتب جديد يتم عبر لوحة الأدمن فقط. الأدمن يدخل: اسم المكتب، رقم الهاتف، المحافظة، المنطقة. يضغط "إضافة"، فيتولد تلقائياً: رقم معرف المكتب (ID) وكلمة مرور عشوائية.
بيانات الاعتماد رقم ID (مثل: OF-001) + كلمة المرور. ترسل هذه البيانات إلى صاحب المكتب عبر الواتساب أو الإيميل.
شاشة الدخول صفحة دخول خاصة بالمكاتب (/office/login). حقلان: رقم المكتب + كلمة المرور.
تغيير كلمة المرور بعد أول دخول، يُجبر النظام المكتب على تغيير كلمة المرور إلى كلمة سرية خاصة به.
حماية الجلسة لا يستطيع أحد دخول لوحة مكتب إلا ببياناته. كل شيء يسجل باسم المكتب.

ب. لوحة تحكم المكتب (بعد الدخول)

1. إدارة العقارات

الميزة الوصف
إضافة عقار جديد نموذج متكامل: نوع العقار (أرض/شقة/دار/محل)، المحافظة، المنطقة، السعر، المساحة، عدد الغرف، الوصف، الصور (حتى 10 صور)، فيديو (رابط يوتيوب اختياري).
قائمة عقاراتي جدول بكل العقارات التي أضافها المكتب: صورة مصغرة، العنوان، السعر، الحالة (متاح/معلق/مباع)، تاريخ الإضافة.
تعديل عقار إمكانية تعديل أي بيانات للعقار.
حذف عقار إمكانية حذف العقار من قاعدة بيانات الشبكة.
تغيير حالة العقار تغيير سريع: متاح ← معلق ← مباع. العقار المباع يختفي من البحث العام لكن يبقى في سجل المكتب.
رفع تقرير الفحص رفع تقرير الفحص الذهبي (PDF) الخاص بالعقار. حتى يُرفع التقرير، يبقى العقار "قيد التدقيق" ولا يظهر للجمهور.

2. البحث عن عقارات الشبكة

الميزة الوصف
محرك بحث متقدم بحث بكل العقارات المضافة من كل المكاتب: تصفية حسب النوع، المحافظة، المنطقة، السعر (من-إلى)، المساحة.
نتائج البحث عرض العقارات المطابقة. يستطيع المكتب رؤية تفاصيل أي عقار وبيانات المكتب صاحب العقار.
طلب وساطة زر "اطلب الوساطة" على عقار ليس له. يرسل إشعاراً للمكتب المالك. إذا تمت الصفقة، تُحسب العمولة تلقائياً.
إحالة زبون زر "أحل زبوناً". يملأ نموذجاً: اسم الزبون، رقمه، ملاحظات. ترسل للمكتب المالك. إذا تمت الصفقة، يُدفع 100 دولار للمُحيل.

3. العقود والعمولات

الميزة الوصف
قائمة صفقاتي سجل بكل صفقة تمت عبر المكتب: رقم الصفقة، العقار، المشتري، السعر، تاريخ الإتمام، حالته.
حالة الصفقة قيد التفاوض ← عقد موقع ← قيد نقل الملكية ← مكتملة ← ملغاة.
حساب العمولة لكل صفقة مكتملة، يظهر: قيمة العقار، نسبة عمولة المكتب، عمولة الشبكة، صافي عمولة المكتب.
طلب صرف عمولة زر لتقديم طلب صرف العمولة المستحقة للمكتب. يظهر تنبيه "قيد المراجعة" حتى يصرفه الأدمن.

4. التواصل والإشعارات

الميزة الوصف
الإشعارات أي طلب وساطة، أو إحالة زبون، أو تغيير حالة صفقة، يصل كإشعار فوري داخل لوحة المكتب.
محادثة داخلية (Chat) شات بسيط بين المكاتب للاستفسار عن عقار.
قائمة المحامين المعتمدين يظهر للمكتب قائمة بمحامي الشبكة: الاسم، التخصص، رقم الهاتف. يستطيع طلب خدمة منهم.
قائمة ورش الصيانة قائمة الورش المعتمدة (كهربائي، سباك، إلخ) مع أرقامهم وتقييماتهم.

ثانياً: قائمة ميزات شبكة المحامين

أ. التسجيل والدخول

الميزة الوصف التقني
تسجيل محامٍ جديد يتم عبر لوحة الأدمن فقط. الأدمن يدخل: اسم المحامي، رقم الهاتف، البريد الإلكتروني، التخصص (عقاري/إداري/تجاري)، المحافظة. يتولد تلقائياً: رقم معرف المحامي (L-ID) وكلمة مرور عشوائية.
بيانات الاعتماد رقم ID (مثل: LW-001) + كلمة المرور. ترسل إلى المحامي.
شاشة الدخول صفحة دخول خاصة بالمحامين (/lawyer/login). حقلان: رقم المحامي + كلمة المرور.

---

ب. لوحة تحكم المحامي (بعد الدخول)

1. الملف الشخصي المهني

الميزة الوصف
تعديل الملف إضافة/تعديل: الصورة الشخصية، سنوات الخبرة، التخصصات الفرعية، رقم الإجازة، رقم النقابة، عنوان المكتب.
حالة الاستلام زر: "متاح لاستقبال طلبات" / "مشغول". يحدد للمكاتب إن كان يستقبل فحوصات جديدة أم لا.
تقييماتي نجمة من 5 + تعليقات المكاتب والزبائن عليه.

2. طلبات الفحص القانوني (الخدمة الأساسية)

الميزة الوصف
قائمة الطلبات الجديدة أي طلب فحص يرسله مكتب أو زبون يظهر هنا: اسم الطالب، رقم العقار، نوع الفحص المطلوب (فضي/ذهبي/ماسي)، التاريخ.
قبول أو رفض زر "قبول" ليبدأ العمل. زر "رفض" مع إبداء السبب.
